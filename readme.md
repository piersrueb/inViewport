### JS In Viewport Function

Loops through your page's section tags and adds the in ``` in-viewport ``` class when they pass the fold.

Demo <a href="https://codepen.io/Rueb/pen/xjNMWL">here</a>.

   ((((^))  /  \
 .-')))(((-'   /
    (((()) __/'
